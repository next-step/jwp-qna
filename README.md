# JPA

## 3단계 : 질문 삭제하기 리팩터링

### 기능 요구 사항

- 질문 데이터를 삭제 할때, 상태만 삭제 상태로 변경
- 삭제 가능 케이스 정리
  - 로그인 사용자와 질문한 사람이 같은 경우
  - 답변이 없는 경우
  - 질문자와 답변 글의 모든 답변자가 같은 경우
- 삭제 불가능 케이스 정리
  - 질문자와 답변자가 다른 경우 답변 삭제 불가
- 질문 삭제할 때, 답변 또한 삭제하도록 구현
- 질문과 답변 삭제 이력에 대한 정보를 DeleteHistory를 활용해 남김
